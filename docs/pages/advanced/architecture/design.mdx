# Design Principles

See [Architecture Overview](/advanced/architecture/overview) for detailed design information.

## Core Principles

1. **Data-Oriented Design** - Cache-conscious layouts
2. **Zero-Cost Abstractions** - Compile-time optimization
3. **Explicit Memory Management** - No hidden allocations
4. **Performance First** - Optimized hot paths
