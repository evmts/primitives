# Core Types

## EvmConfig

Compile-time configuration for the EVM.

```zig
const g = @import("guillotine");
const cfg = g.EvmConfig{
  .eips = .{ .hardfork = .CANCUN },
  .max_call_depth = 1024,
  .max_input_size = 131072,
  .enable_precompiles = true,
  .enable_fusion = true,
  .stack_size = 1024,
  .memory_initial_capacity = 4096,
  .memory_limit = 0xFFFFFF,
  .block_gas_limit = 30_000_000,
};
const Evm = g.Evm(cfg);
```

## Hardfork

Enumeration of supported forks with helpers.

```zig
const hf = g.Hardfork.CANCUN;
_ = hf.isAtLeast(.BERLIN);
```

## BlockInfo

Block context for environment opcodes.

```zig
var block = g.BlockInfo.init();
block.number = 18_000_000;
block.timestamp = 1_700_000_000;
block.coinbase = g.Primitives.ZERO_ADDRESS;
block.base_fee = 0;
```

## TransactionContext

Transaction-level context (constant during a call).

```zig
const tx = g.TransactionContext{
  .gas_limit = 3_000_000,
  .coinbase = g.Primitives.ZERO_ADDRESS,
  .chain_id = 1,
  .blob_versioned_hashes = &.{},
  .blob_base_fee = 0,
};
```

## CallParams

Per-operation parameters (specialized per EVM config).

```zig
const CallParams = Evm.CallParams; // union(enum)
// tags: .call, .callcode, .delegatecall, .staticcall, .create, .create2

// example
const p = CallParams{ .call = .{ .caller = origin, .to = target, .value = 0, .input = calldata, .gas = 500_000 } };

// helpers
_ = p.getGas();
_ = p.getCaller();
_ = p.getInput();
```

## CallResult

Execution result value.

```zig
const r = evm.call(p);
if (r.success) {
  // r.output holds return data
} else {
  // r.output may contain revert data; r.error_info may contain diagnostics
}

const used = p.getGas() -| r.gas_left; // saturating subtraction
```
