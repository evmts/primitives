## Features

- **Performance First** - Optimized for speed with minimal allocations
- **Type Safety** - Strong typing throughout the codebase
- **Memory Efficient** - Cache-conscious struct layout
- **Modular Design** - Clean separation of concerns
- **Comprehensive Testing** - Differential testing against revm

## Quick Start

```bash
# Clone the repository
git clone https://github.com/williamcory/Guillotine.git
cd Guillotine

# Build the project
zig build

# Run tests
zig build test
```

## Architecture

Guillotine follows a modular architecture with clear separation between:

- **EVM Core** - Opcode execution and stack management
- **State Management** - Database, journaling, and access lists
- **Bytecode Analysis** - Optimization and jump table generation
- **Precompiles** - Built-in contract implementations
